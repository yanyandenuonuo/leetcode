# Reverse Linked List

## Description

&emsp;&emsp;Reverse a singly linked list.

## Example

### eg1

```
    Input: 1->2->3->4->5->NULL
    
    Output: 5->4->3->2->1->NULL
```

## Follow up

&emsp;&emsp;A linked list can be reversed either iteratively or recursively. Could you implement both?
