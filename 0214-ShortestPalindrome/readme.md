# Shortest Palindrome

## Description

&emsp;&emsp;Given a string **s**, you are allowed to convert it to a palindrome by adding characters in front of it. 
Find and return the shortest palindrome you can find by performing this transformation.

## Example

### eg1

```
    Input: "aacecaaa"
    
    Output: "aaacecaaa"
```

### eg2

```
    Input: "abcd"
    
    Output: "dcbabcd"
```
