# Bitwise AND of Numbers Range

## Description

&emsp;&emsp;Given a range \[m, n\] where 0 \<\= m \<\= n \<\= 2147483647, return the bitwise AND of all numbers in this 
range, inclusive.

## Example

### eg1

```
    Input: [5,7]
    
    Output: 4
```

### eg2

```
    Input: [0,1]
    
    Output: 0
```
