# Remove Linked List Elements

## Description

&emsp;&emsp;Remove all elements from a linked list of integers that have value **val**.

## Example

### eg1

```
    Input:  1->2->6->3->4->5->6, val = 6
    
    Output: 1->2->3->4->5
```

## Difficulty

&emsp;&emsp;Easy
